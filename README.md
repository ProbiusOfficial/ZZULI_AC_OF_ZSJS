### 时间魔法

用控制台延迟时间，F12后进入控制台，控制台中输入min = 40，可重置倒计时，
```
min = xx
```

min只是用来控制autoSubmit()函数，min<=0时会执行自动提交

而要注意的是，提交的时候使用的 "starttime" 来校验是否过短或者超时答题，请确保提交的时候，starttime 和现在的时间差大于15分钟。
```
starttime = 'xxxx-xx-xx xx:xx:xx'
```
